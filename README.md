# Rust 神经网络项目

本项目使用 Rust 语言实现神经网络相关功能，主要依赖以下开源库：

---

## 📦 主要依赖
- [tch](https://crates.io/crates/tch)：Rust 的 PyTorch 封装，提供深度学习相关的张量操作、模型构建与训练等能力。
- [plotters](https://crates.io/crates/plotters)：用于绘图和可视化（如需要展示训练过程或结果）。

---

## 🛠️ 运行方法
1. 安装 Rust 工具链（建议 nightly）
2. 安装依赖：
   ```shell
   cargo build
   ```
3. 运行：
   ```shell
   cargo run --release
   ```
